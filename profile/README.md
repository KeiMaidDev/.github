<h2 align="center">Kei_Maid</h2>

<p align="center">支持多款音游成绩查询、ba游戏攻略及娱乐功能的QQ机器人</p>

## 介绍

本项目基于[KoiShi](https://koishi.chat)及[QQ开放平台](https://q.qq.com)开发，提供音游、ba游戏攻略、点歌、漂流瓶等机器人功能

## 项目结构

存储库内存储了KeiMaidDev项目团队开发的适用于[adapter-qq-crack](https://github.com/koishi-shangxue-plugins/koishi-plugin-adapter-qq-crack)适配器的插件
